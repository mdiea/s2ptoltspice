# s2ptoltspice
Converts two-port Touchstone files to **LTspice** subcircuit.

Supports three subcommands:

- **`convert`** – Convert a full S2P file into LTspice files.
- **`merge`** – Merge two NanoVNA half-measurements (forward + reverse) into a complete S2P file.
- **`merge-convert`** – Merge two half-measurements and immediately convert the result to LTspice.

Output files generated by `convert` / `merge-convert`:
- `[name].sub`: The subcircuit
- `[name].asy`: The circuit symbol
- `test-[name].asc`: An LTspice circuit example using the model

---

## Usage

```
usage: s2ptoltspice.exe [-h] [--version] COMMAND ...

NanoVNA S2P → LTspice model toolkit

options:
  -h, --help  show this help message and exit
  --version   show program's version number and exit

COMMAND:
  convert       Convert a full S2P file to LTspice .sub / .asy / .asc
  merge         Merge two NanoVNA half-measurements into a full S2P
  merge-convert Merge two half-measurements and convert the result to LTspice
```

### convert

```
usage: s2ptoltspice.exe convert [-h] [--fmin FMIN] [--fmax FMAX] [--overwrite] [--silent] s2p_file

positional arguments:
  s2p_file       Full S2P Touchstone file to convert

options:
  --fmin FMIN    Min frequency (e.g. 1MHz)
  --fmax FMAX    Max frequency (e.g. 500MHz)
  --overwrite    Overwrite existing files without asking (default: False)
  --silent       Suppress output messages (default: False)
```

### merge

```
usage: s2ptoltspice.exe merge [-h] [-o OUTPUT] [--silent] forward reverse

positional arguments:
  forward          S2P with S11 and S21 (port1→port2)
  reverse          S2P with S22 and S12 (port2→port1, DUT flipped)

options:
  -o, --output OUTPUT  Output file (default: <forward_stem>_full.s2p)
  --silent             Suppress output messages (default: False)
```

### merge-convert

```
usage: s2ptoltspice.exe merge-convert [-h] [-o OUTPUT] [--fmin FMIN] [--fmax FMAX] [--overwrite] [--silent] forward reverse

positional arguments:
  forward          S2P with S11 and S21 (port1→port2)
  reverse          S2P with S22 and S12 (port2→port1, DUT flipped)

options:
  -o, --output OUTPUT  Intermediate merged S2P file (default: <forward_stem>_full.s2p)
  --fmin FMIN          Min frequency (e.g. 1MHz)
  --fmax FMAX          Max frequency (e.g. 500MHz)
  --overwrite          Overwrite existing files without asking (default: False)
  --silent             Suppress output messages (default: False)
```

## Examples

### Convert a full S2P file

```
>s2ptoltspice.exe convert network_example.s2p
Processing file: network_example.s2p
Filtered frequencies: from 100000.0Hz to 500000000.0Hz
Z0 port 1: 50.0   Z0 port 2: 50.0
Write network_example.sub
Write network_example.asy
Write test-network_example.asc
```

### Merge two half-measurements

```
>s2ptoltspice.exe merge filter.s2p filterInv.s2p -o filter_full.s2p
Written: filter_full.s2p
  Frequencies : 100000 Hz  →  500000000 Hz  (101 points)
  S11 |avg|   : 0.4123
  S21 |avg|   : 0.5871
  S12 |avg|   : 0.5869
  S22 |avg|   : 0.4118
```

### Merge and convert in one step

```
>s2ptoltspice.exe merge-convert filter.s2p filterInv.s2p -o xtal_filter
Merged S2P written: xtal_filter.s2p
Filtered frequencies: from 100000.0Hz to 500000000.0Hz
Write xtal_filter.sub
Write xtal_filter.asy
Write test-xtal_filter.asc
```
